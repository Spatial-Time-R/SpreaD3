h1. PURPOSE

SPREAD 2 is a program supporting processing and visualizations of Bayesian phylogeographic models.

h3. DISCRETE TREE PARSING

h3. DISCRETE TREE KML RENDERING

* use all default settings
-render kml -json /home/filip/Dropbox/JavaProjects/Spread2/d3renderer/src/data/ebov_discrete.json -output test.kml

* map point area attribute to a continuous trait
-render kml -json /home/filip/Dropbox/JavaProjects/Spread2/d3renderer/src/data/ebov_discrete.json -pointAreaMapping height -output test.kml

* map point area attribute to a discrete trait
-render kml -json /home/filip/Dropbox/JavaProjects/Spread2/d3renderer/src/data/ebov_discrete.json -pointAreaMapping location -output test.kml

* map point color attribute to a continuous trait
-render kml -json /home/filip/Dropbox/JavaProjects/Spread2/d3renderer/src/data/ebov_discrete.json -pointAreaMapping height -pointColorMapping height -output test.kml

* map point color attribute to a discrete trait
-render kml -json /home/filip/Dropbox/JavaProjects/Spread2/d3renderer/src/data/ebov_discrete.json -pointAreaMapping height -pointColorMapping location -output test.kml

* override default area color
-render kml -json /home/filip/Dropbox/JavaProjects/Spread2/d3renderer/src/data/ebov_discrete.json -pointAreaMapping height -pointColorMapping location -areacolor 0 255 255 -output test.kml
